

1. install java in ubuntu 
sudo apt install openjdk-17-jre-headless 

2. install maven in ubuntu
apt install maven -y   # 

3. Install jenkins LTS version- befor installing jenkins, java must be installed.
sudo wget -O /usr/share/keyrings/jenkins-keyring.asc \
  https://pkg.jenkins.io/debian-stable/jenkins.io-2023.key
echo deb [signed-by=/usr/share/keyrings/jenkins-keyring.asc] \
  https://pkg.jenkins.io/debian-stable binary/ | sudo tee \
  /etc/apt/sources.list.d/jenkins.list > /dev/null
sudo apt-get update
sudo apt-get install jenkins

sudo systemctl enable jenkins
sudo systemctl start jenkins
sudo systemctl status jenkins

Now you have a working MySQL database setup on Ubuntu.
CREATE USER 'adi'@'localhost' IDENTIFIED BY 'Test@123';
Creates a new MySQL account adi that can log in only from the same machine.
By default on MySQL 8, this uses the caching_sha2_password auth plugin.

ALTER USER 'adi'@'localhost' IDENTIFIED WITH mysql_native_password BY 'Test@123';
Changes the auth plugin for adi@localhost to mysql_native_password (keeps the same password).

This is often used to avoid the JDBC error “Public Key Retrieval is not allowed” without adding allowPublicKeyRetrieval=true to your JDBC URL.
